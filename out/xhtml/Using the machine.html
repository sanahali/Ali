Using the Ice Cream Machine in Shanks Hall

Who can use the machine and when can it be used?

This ice cream machine is for all English students, faculty, and maintenance staff to enjoy. We ask that you be courteous and patient as there may be a lot of people wanting to use the machine at once.

Users are allowed to make one batch of ice cream at a time. If no one else is waiting for the machine, you may make more. However, if there are others waiting to use the machine, you will have to get back in line to make a second batch.

Contact in case of problem
